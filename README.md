## Rock | Paper | Scissor

Rock|Paper|Scissor game is a simple fun game in which both the players have to make a rock, paper, or scissors. It has only two possible outcomes a draw, or a win for one player and a loss for the other player.The player has to choose one option among the rock, paper, and scissors. ## Project Specifications

- Player vs Computer game.

  - _Lose_ = -1 pts
  - _Draw_ = 0 pts
